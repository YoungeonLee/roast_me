HOW TO PLAY

Set Profile
  • Click "Set Profile" button or here
  • Pick a username
  • Upload a picture
  • Write a description
Join or Create Game
  • Join or create game
  • The game requires 4+ people
Roast Each Other!
  • The player who reaches the goal score first wins
  • Roast the roastee
  • The judge will decide the winner of the round
  • The winner will judge the next round
  • And the next roastee will be randomly picked
  • For the first round and skipped rounds, roles will be randomly assigned
